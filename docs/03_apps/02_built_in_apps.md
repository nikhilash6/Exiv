# Built-in Apps

Exiv includes several powerful apps out-of-the-box inside the `apps/` directory.

## Examples of Included Apps
*   **`wan_video.py`**: A simple text-to-video app using the WAN model.
*   **`wan_animate.py`**: A use case of WAN Animate for replacing characters in videos.

## Running Built-in Apps
To run any of the built-in applications natively:

```bash
python apps/wan_video.py
```
Check the specific app files for their required CLI arguments or environment variables.
