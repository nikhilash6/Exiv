# Built-in Apps

Exiv includes several powerful apps out-of-the-box inside the `apps/` directory.

## Examples of Included Apps
*   **`simple_t2v.py`**: A simple text-to-video app using the WAN model.
*   **`character_replace/app.py`**: Character replacement and animation using WAN Animate.
*   **`frame_interpolation/app.py`**: Frame interpolation between keyframes using WAN.
*   **`qwen3_tts/app.py`**: Qwen3 Text-to-Speech with voice cloning.

## Running Built-in Apps
To run any of the built-in applications natively:

```bash
python apps/simple_t2v.py
```
Check the specific app files for their required CLI arguments or environment variables.
