# Overview

Welcome to the Exiv documentation! Exiv is a fast, lightweight, and extensible AI backend engine designed for running, patching, and serving generative models and orchestrating workflows efficiently. 

This documentation is structured to help you go from setup to building your own apps and extensions quickly:

*   **[Getting Started](./02_installation.md)**: Set up the environment and run your first app.
*   **[Operational Modes](./04_modes.md)**: Explore the different ways to run and integrate Exiv.
*   **[Configuration](../02_configuration/01_dynamic_offloading.md)**: Learn how to manage VRAM usage, model paths, and logging.
*   **[Apps](../03_apps/01_overview.md)**: Discover built-in apps or build your own high-level interfaces to run models.
*   **[Internals](../04_internals/01_model_mixin.md)**: Dive deep into how Exiv manages models, LoRAs, hooks, and the API server.
*   **[Extensions](../05_extensions/01_overview.md)**: Extend the engine's core capabilities by writing custom plugins.
*   **[Development](../06_development/01_contribution_guide.md)**: Guidelines for contributing code and running the test suite.
